# Рабочее пространство Leva4ka

Личное рабочее пространство с пет-проектами и экспериментами.

## Структура

- `teachme/` - учебный проект.
- `cool-site/`, `spline-site/`, `spline-app/`, `splash-site/` - отдельные веб-проекты.
- `drafts/website-playground/` - отдельные черновики на HTML/JS/Vue и связанные ассеты.
- `data/zhkh/` - файлы с данными (CSV/JSON).

## Примечания

- В репозитории используется корневой `.gitignore`, чтобы не хранить в git зависимости и сгенерированные файлы.
- В `splash-site/` сейчас есть локальные изменения, они сохранены как есть.

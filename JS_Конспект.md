# 📚 Конспект по JavaScript

## Урок 1: Переменные и типы данных

### 📖 ТЕОРИЯ
**Что такое переменная?**
Переменная — это как коробка, в которой мы хранишь информацию.

**Три способа создать коробку:**
- `var` — старый способ (не используем)
- `let` — нормальный способ (рекомендуется)
- `const` — неизменяемая коробка (когда не хочешь менять)

### 📝 ТИПЫ ДАННЫХ

1. **Строки (String)** - текст
   - `"Привет"` или `'Привет'` или `` `Привет` ``

2. **Числа (Number)** - просто числа
   - `42`, `3.14`, `-5`

3. **Логические (Boolean)** - ДА или НЕТ
   - `true` или `false`

4. **Специальные:**
   - `null` — "ничего"
   - `undefined` — "не определено"

## Урок 2: Математика в JavaScript (операции)

### 📖 ТЕОРИЯ
**Волшебник тоже может считать!**

JavaScript может складывать, вычитать, умножать, делить числа.

**Основные операции:**
- `+` плюс (сложение)
- `-` минус (вычитание)
- `*` звёздочка (умножение)
- `/` слэш (деление)
- `%` остаток от деления
- `**` возведение в степень

### 💪 ПРИМЕРЫ И УПРАЖНЕНИЯ

**Упражнение 1: Сумма**
```javascript
let яблоки = 5;
let груши = 3;
let всегоФруктов = яблоки + груши;
console.log(всегоФруктов);  // → 8
```

**Упражнение 2: Вычитание**
```javascript
let деньги = 1000;
let потратил = 250;
let осталось = деньги - потратил;
console.log(осталось);  // → 750
```

**Упражнение 3: Умножение**
```javascript
let цена = 150;
let количество = 4;
let сумма = цена * количество;
console.log(сумма);  // → 600
```

## Урок 3: Условия (if/else)

### 📖 ТЕОРИЯ
**Волшебник принимает решения!**

Условия позволяют коду делать ВЫБОР:
- **Если** условие верно → выполни один код
- **Иначе** → выполни другой код

**Структура:**
```javascript
if (УСЛОВИЕ) {
    // код если верно
} else {
    // код если неверно
}
```

### 📊 СРАВНЕНИЕ (Символы для проверки)

| Символ | Значение | Пример | Результат |
|--------|----------|--------|-----------|
| `>` | больше | `10 > 5` | `true` |
| `<` | меньше | `3 < 10` | `true` |
| `>=` | больше или равно | `5 >= 5` | `true` |
| `<=` | меньше или равно | `3 <= 5` | `true` |
| `==` | равно | `5 == 5` | `true` |
| `!=` | не равно | `5 != 3` | `true` |

### 💪 ПРИМЕРЫ

**Пример 1: Простой if**
```javascript
let возраст = 18;
if (возраст >= 18) {
    console.log("Ты взрослый!");
}
```

**Пример 2: if/else**
```javascript
let деньги = 200;
let цена = 150;

if (деньги >= цена) {
    console.log("Денег хватает!");
} else {
    console.log("Мало денег!");
}
```

**Пример 3: if/else if/else (несколько условий)**
```javascript
let оценка = 85;

if (оценка >= 90) {
    console.log("Отлично!");
} else if (оценка >= 70) {
    console.log("Хорошо!");
} else {
    console.log("Надо учиться!");
}
```

## Урок 4: Циклы (for и while)

### 📖 ТЕОРИЯ

**Цикл** — повторение кода много раз.

**Два типа циклов:**

#### FOR (когда знаешь сколько раз)
```javascript
for (НАЧАЛО; УСЛОВИЕ; ИЗМЕНЕНИЕ) {
    // код что повторяется
}
```

#### WHILE (когда не знаешь сколько раз)
```javascript
while (УСЛОВИЕ) {
    // код
    // ВАЖНО: менять переменную!
}
```

### 💪 ПРИМЕРЫ

**FOR — числа от 1 до 5:**
```javascript
for (let i = 1; i <= 5; i++) {
    console.log(i);
}
```

**FOR — таблица умножения:**
```javascript
let число = 5;
for (let i = 1; i <= 10; i++) {
    console.log(`${число} * ${i} = ${число * i}`);
}
```

**FOR — сумма:**
```javascript
let сумма = 0;
for (let i = 1; i <= 10; i++) {
    сумма = сумма + i;
}
console.log(сумма);  // → 55
```

**WHILE — пароль:**
```javascript
let пароль = "123";
let введённый = "";

while (введённый !== пароль) {
    введённый = prompt("Пароль:");
}
```

### 📋 ДОМАШКА № 4
✅ **СДАНА!**
- ✅ Задача 1: Числа от 1 до 10 (for)
- ✅ Задача 2: Таблица умножения (for)
- ✅ Задача 3: Сумма от 1 до 10 (результат 55)

### 📝 ВАЖНЫЕ МОМЕНТЫ
- `console.log()` ВНУТРИ цикла выводит на каждой итерации
- `console.log()` ЗА циклом выводит один раз
- ВСЕГДА используй `===` и `!==` (не `==` и `!=`)
- В while всегда меняй переменную (иначе бесконечный цикл!)
- `${переменная}` это встраивание переменной в backtick строку

## Урок 5: Функции

### 📖 ТЕОРИЯ

**Что такое функция?**
Функция — это коробка с кодом, которую можно использовать много раз.

Вместо того чтобы писать одно и то же 100 раз:
```javascript
console.log("Привет!");
console.log("Привет!");
console.log("Привет!");
```

Пишешь функцию 1 раз и используешь её:
```javascript
function greet() {
    console.log("Привет!");
}
greet();  // Привет!
greet();  // Привет!
greet();  // Привет!
```

**Структура функции:**
```javascript
function имяФункции(параметр1, параметр2) {
    // КОД функции
    return результат;  // что вернуть (опционально)
}

// Вызов:
имяФункции(значение1, значение2);
```

**Части функции:**
- `function` — ключевое слово
- `имя` — название функции
- `(параметры)` — входные данные (может быть 0, 1 или много)
- `{}` — тело функции
- `return` — что функция выдаёт (опционально)

### 📊 РАЗНИЦА: ЦИКЛ vs ФУНКЦИЯ

| | Цикл | Функция |
|---|---|---|
| **Когда работает** | Когда код доходит до цикла (выполняет N раз подряд) | Когда ты её вызываешь (можно много раз в разных местах) |
| **Для чего** | Повторить один кусок кода N раз сейчас | Переиспользовать код в разных местах программы |

### 💪 5 ТИПОВ ФУНКЦИЙ

**1️⃣ Функция без параметров (просто выполняет код)**
```javascript
function greet() {
    console.log("Привет!");
}

greet();  // → Привет!
greet();  // → Привет! (используем снова!)
```

**2️⃣ Функция с одним параметром**
```javascript
function sayName(name) {
    console.log("Твоё имя: " + name);
}

sayName("Дмитрий");  // → Твоё имя: Дмитрий
```

**3️⃣ Функция с `return` (возвращает результат)**
```javascript
function sum(a, b) {
    return a + b;
}

let результат = sum(5, 3);
console.log(результат);  // → 8
```

**⚠️ ВАЖНО:** `return` vs `console.log()`
```javascript
// НЕПРАВИЛЬНО (console.log вместо return):
function multiply(x, y) {
    console.log(x * y);  // выведет в консоль
}

let число = multiply(4, 7);
console.log(число);  // → undefined (ничего не вернулось!)

// ПРАВИЛЬНО (return):
function multiply(x, y) {
    return x * y;  // возвращает значение
}

let число = multiply(4, 7);
console.log(число);  // → 28
```

**Разница:**
- `return` — возвращает значение из функции (можно потом использовать)
- `console.log()` — выводит в консоль для просмотра (результат не отправляется)

**4️⃣ Функция с несколькими параметрами**
```javascript
function calculatePrice(price, quantity) {
    let total = price * quantity;
    return total;
}

console.log(calculatePrice(100, 5));  // → 500
```

**5️⃣ Функция с условиями внутри**
```javascript
function checkAge(age) {
    if (age >= 18) {
        return "Взрослый";
    } else {
        return "Несовершеннолетний";
    }
}

console.log(checkAge(25));  // → Взрослый
```

## Урок 6: Объекты

### 📖 ТЕОРИЯ

**Что такое объект?**
Объект — это контейнер для группировки связанных данных (как словарь).

Вместо много переменных:
```javascript
let userName = "Дмитрий";
let userAge = 25;
let userCity = "Москва";
```

Используешь объект:
```javascript
let user = {
  name: "Дмитрий",
  age: 25,
  city: "Москва"
};
```

**Структура объекта:**
```javascript
let obj = {
  ключ1: значение1,
  ключ2: значение2,
  ключ3: значение3
};
```

### 🔓 ДОСТУП К СВОЙСТВАМ

**Способ 1: Точечная нотация**
```javascript
const person = { name: "John", age: 25 };
console.log(person.name);   // "John"
console.log(person.age);    // 25
```

**Способ 2: Квадратные скобки**
```javascript
console.log(person["name"]);  // "John"

// Полезно когда имя в переменной:
const key = "name";
console.log(person[key]);  // "John"
```

### ⚙️ МЕТОДЫ ОБЪЕКТА (функции внутри объекта)

```javascript
const calculator = {
  name: "My Calculator",
  lastResult: 0,
  
  // Метод (функция внутри объекта)
  add: function(a, b) {
    this.lastResult = a + b;
    return this.lastResult;
  },
  
  multiply: function(a, b) {
    this.lastResult = a * b;
    return this.lastResult;
  }
};

console.log(calculator.add(5, 3));        // 8
console.log(calculator.multiply(4, 7));   // 28
```

**`this`** — это ссылка на текущий объект. `this.lastResult` = свойство этого объекта.

### 💪 ПРИМЕРЫ

**Реальный пример: Товар**
```javascript
const laptop = {
  brand: "Lenovo",
  model: "ThinkPad",
  price: 1000,
  color: "black",
  
  getInfo: function() {
    return `${this.brand} ${this.model} - $${this.price}`;
  }
};

console.log(laptop.getInfo());  // "Lenovo ThinkPad - $1000"
```

### 📝 КЛЮЧЕВЫЕ МОМЕНТЫ
- **Объект** = группировка связанных данных
- **Доступ:** `object.property` или `object["property"]`
- **Методы** = функции внутри объекта
- **`this`** = ссылка на текущий объект
- Все реальные данные (API, БД) структурированы как объекты!

## Урок 7: Массивы

### 📖 ТЕОРИЯ

**Что такое массив?**
Массив — это список данных в одной переменной.

Вместо много переменных:
```javascript
let фрукт1 = "яблоко";
let фрукт2 = "банан";
let фрукт3 = "апельсин";
```

Используешь массив:
```javascript
const fruits = ["яблоко", "банан", "апельсин"];
```

**Структура массива:**
```javascript
let arr = [элемент0, элемент1, элемент2, элемент3];
```

### 📏 ИНДЕКСАЦИЯ (ОЧЕНЬ ВАЖНО!)

**В JavaScript отсчёт начинается с 0, НЕ с 1!**

```javascript
const fruits = ["яблоко", "банан", "апельсин"];

console.log(fruits[0]);   // "яблоко" (первый)
console.log(fruits[1]);   // "банан" (второй)
console.log(fruits[2]);   // "апельсин" (третий)
console.log(fruits[3]);   // undefined (нет 4-го элемента)
```

### 📊 РАЗМЕР МАССИВА

```javascript
const fruits = ["яблоко", "банан", "апельсин"];
console.log(fruits.length);  // 3
```

### ➕ МЕТОДЫ МАССИВА (самые важные)

**push() — добавить в конец**
```javascript
const fruits = ["яблоко", "банан"];
fruits.push("апельсин");
console.log(fruits);  // ["яблоко", "банан", "апельсин"]
```

**pop() — удалить с конца**
```javascript
const fruits = ["яблоко", "банан", "апельсин"];
const last = fruits.pop();
console.log(last);     // "апельсин"
console.log(fruits);   // ["яблоко", "банан"]
```

**shift() — удалить с начала**
```javascript
const fruits = ["яблоко", "банан", "апельсин"];
const first = fruits.shift();
console.log(first);    // "яблоко"
console.log(fruits);   // ["банан", "апельсин"]
```

**unshift() — добавить в начало**
```javascript
const fruits = ["банан", "апельсин"];
fruits.unshift("яблоко");
console.log(fruits);  // ["яблоко", "банан", "апельсин"]
```

**forEach() — перебрать все элементы**
```javascript
const fruits = ["яблоко", "банан", "апельсин"];

fruits.forEach(function(fruit) {
  console.log(fruit);
});

// Выведет:
// яблоко
// банан
// апельсин
```

### 💪 ПРИМЕРЫ

**Массив объектов (ОЧЕНЬ ЧАСТО!)**
```javascript
const user = [
   { name: "John", age: 20, isActive: true},
   { name: "Jane", age: 21, isActive: true},
   { name: "Jim", age: 22, isActive: false},
];

// Добавить нового пользователя
user.push({ name: "Jill", age: 23, isActive: true});

// Перебрать всех
user.forEach(function(person) {
   console.log(`${person.name} is ${person.age} years old`);
});

// Только активные
user.forEach(function(person) {
   if (person.isActive) {
      console.log(`${person.name} is active`);
   }
});
```
